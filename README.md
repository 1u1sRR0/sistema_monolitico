
# Travel Assistant IA

Asistente inteligente de viajes desarrollado en Node.js con Fastify. Permite consultar clima, ubicaciones y obtener recomendaciones mediante integración con APIs externas y modelos de lenguaje.

---

## Funcionalidades

- Consulta de clima en tiempo real (OpenWeather)
- Geolocalización de ciudades (OpenStreetMap)
- Respuestas inteligentes con LLM (Gemini y Ollama)
- Bot de Telegram
- Autenticación con JWT
- Historial y estadísticas de uso
- API documentada con Swagger

---

## Tecnologías

- Node.js + Fastify
- PostgreSQL
- Docker
- Telegram Bot API
- OpenWeather API
- OpenStreetMap (Nominatim)
- Gemini API
- Ollama

---

npm run dev
